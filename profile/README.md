# Luminarios

O **Luminarios** é um sistema de gerenciamento de livros projetado para oferecer uma experiência de catalogação de alta qualidade, robusta e totalmente focada no usuário. Somos especializados no desenvolvimento de soluções de software que aprimoram a organização, o controle de inventário e a acessibilidade a acervos literários, atendendo com precisão desde coleções pessoais complexas até bibliotecas comunitárias ou acadêmicas.

## Nossa Filosofia

Acreditamos que os livros são os maiores faróis de conhecimento da humanidade e que a organização rigorosa é a chave para tornar esse conhecimento verdadeiramente útil e acessível. O Luminarios nasceu da premissa de que gerenciar um acervo não deve ser um processo burocrático, lento ou confuso. Nossa filosofia baseia-se na centralização inteligente de dados e no design intuitivo, eliminando atritos operacionais para que a conexão entre o leitor e a obra seja imediata. Iluminamos a gestão de dados para que a cultura e a informação permaneçam sempre organizadas e preservadas.

## Stack de Tecnologias

O sistema é construído sobre uma arquitetura limpa, estável e escalável, utilizando tecnologias consagradas no mercado de desenvolvimento web:

- **PHP** (Mecanismo central de back-end, responsável pelo processamento de regras de negócio, rotas e a segurança das requisições)
- **MySQL** (Banco de dados relacional principal, configurado para garantir a integridade, persistência e alta disponibilidade dos dados do catálogo)
- **SQLite** (Engine de banco de dados leve e autocontida, utilizada para facilitar o ambiente de desenvolvimento local e portabilidade do sistema)
